Redis
支持的数据类型：String（字符串），list（列表），set（集合），hash（哈希），zset（sorted set：有序集合）。
String（字符串）：基本的数据类型，一个key对应一个value，最大存储512MB；SET key value，GET key。
list（列表）：简单的字符串列表，按照插入顺序排序；lpush/rpush key value,lrange key 0 5(取值范围)。
set（集合）：是String类型的无序集合，集合是通过哈希表实现的，所以添加，删除，查找的复杂度都是 O(1)。sadd key value，添加一个 string 元素到 key 对应的 set 集合中，成功返回 1，如果元素已经在集合中返回 0。smembers key
hash（哈希）：是一个键值(key=>value)对集合。Redis hash 是一个 string 类型的 field 和 value 的映射表，hash 特别适合用于存储对象。
redis 127.0.0.1:6379> DEL ceshi
redis 127.0.0.1:6379> HMSET ceshi field1 "Hello" field2 "World"
"OK"
redis 127.0.0.1:6379> HGET ceshi field1
"Hello"
redis 127.0.0.1:6379> HGET ceshi field2
"World"
zset（sorted set：有序集合）：Redis zset 和 set 一样也是string类型元素的集合,且不允许重复的成员。不同的是每个元素都会关联一个double类型的分数。redis正是通过分数来为集合中的成员进行从小到大的排序。zset的成员是唯一的,但分数(score)却可以重复。zadd 命令添加元素到集合，元素在集合中存在则更新对应score。zadd key score（分数） value ，ZRANGEBYSCORE key 0 1000
